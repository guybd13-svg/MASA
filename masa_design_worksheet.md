## 📐 תוצר 2: מערכת עיצוב (Design System)

### צבעים:

| תפקיד | קוד צבע (Hex / RGBA) | שימוש |
| :--- | :--- | :--- |
| **Primary** | `#9d4edd` (Purple) | סגול ויולט יוקרתי לכפתורים ראשיים, אייקון ניווט פעיל וסימון כרטיסיה נוכחית. |
| **Secondary** | `rgba(20, 22, 37, 0.45)` | רקע כרטיסיות זכוכית שקופות (במפה, בתקציב, ביומן ובצ'קליסט). |
| **Accent** | `#4ea8de` (Sky Blue) | כחול שמיים לסיכות מפה גיאוגרפיות, תוויות מיקומים וקורדינטות. |
| **Background** | `#090a0f` (Dark Navy) | צבע הרקע הכהה והעמוק של כל מסכי האתר המבליט את כרטיסיות הזכוכית. |
| **Text** | `#ffffff` (White) | לבן נקי וקריא במיוחד לכותרות וטקסטים ראשיים. |
| **Error / שגיאה** | `#ff70a6` (Soft Pink) | ורוד רך לחריגות תקציב, כפתורי מחיקה והתרעות על חוסר בנתוני GPS. |
| **Success / הצלחה** | `#52b788` (Emerald Green) | ירוק סולידי לסטטיסטיקות תקציב חיוביות, פריטים שנארזו ומשימות שהושלמו. |

### טיפוגרפיה:

| תפקיד | שם הפונט | גודל / משקל |
| :--- | :--- | :--- |
| **Heading** | **Outfit** | משקל Bold, גודל `24px` עד `32px` עבור כותרות מסכים וקטגוריות. |
| **Body** | **Outfit** | משקל Regular, גודל `16px` עבור תיאורי חוויות ביומן, פריטי ציוד והוצאות. |
| **Caption / Label** | **Outfit** | משקל Medium, גודל `12px` עד `14px` לתוויות תאריכים, מיקומים וקטגוריות. |

### ריווח:

| רכיב ריווח | תיאור |
| :--- | :--- |
| **יחידת בסיס (Base Unit)** | כל הריווחים באפליקציה יהיו כפולות של `8px` (`8px`, `16px`, `24px`, `32px` וכו') לשמירה על סדר ויזואלי הדוק. |
| **עיגול פינות (Border Radius)** | עיגול פינות מודרני של `16px` לכרטיסיות ופופ-אפים, `10px` לכפתורים וטאבים ו-`8px` לכפתורים משניים. |

### סגנון רכיבים:

| רכיב | תיאור סגנון |
| :--- | :--- |
| **כפתורים (Buttons)** | פינות מעוגלות של `10px`, רקע סגול ויולט יוקרתי (`#9d4edd`) עם טקסט לבן מודגש ומעבר עכבר המפעיל הגדלה קלה (scale). כפתורי מחיקה מבוססים על רקע שקוף למחצה והחלפת צבע לורוד רך (`#ff70a6`) בריחוף. |
| **כרטיסים (Cards)** | רקע כהה חצי-שקוף (`rgba(20, 22, 37, 0.45)`) עם אפקט טשטוש רקע עשיר (`blur(20px) saturate(180%)`), פינות מעוגלות של `16px` וגבול דק (`rgba(255, 255, 255, 0.08)`) להפרדה ברורה מהרקע הכהה. |
| **שדות קלט (Inputs)** | רקע זכוכית כהה, פינות מעוגלות של `10px`, גבול שקוף למחצה שהופך לגבול סגול מוגדר ומודגש (`#9d4edd`) בזמן פוקוס/לחיצה. |
| **ניווט (Navigation)** | סרגל עליון (Top Bar) זכוכית מלוטשת ברוחב מלא, בעל אפקט טשטוש רקע (`blur(25px)`) וסטורציה של `200%`. גבול תחתון עדין וצל פנימי ליצירת מראה יוקרתי. כרטיסיות הניווט הפעילות מובלטות ברקע סגול עדין (`rgba(157, 78, 221, 0.12)`) וטקסט סגול (`#9d4edd`). |
